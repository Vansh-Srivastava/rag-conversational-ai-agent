# rag-conversational-ai-agent
Overview

#An intelligent Retrieval-Augmented Generation (RAG) chatbot that combines semantic search and Large Language Models to provide accurate, context-aware responses from custom knowledge bases.

#Features
#->Semantic document retrieval using Pinecone
#->Buffer memory for multi-turn conversations
#->Context-aware response generation
#->Hallucination reduction through document grounding
#->Modular architecture with LangChain
#Tech Stack
#->Python
#->LangChain
#->Pinecone
#->Google Gemini
#->Vector Search
#->n8n
